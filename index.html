Shade Game

A Shady puzzle game for iPhones

View the Project on GitHub jrheaw/ShadeGame

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Shade Game is a puzzle game in which you try to shade all of the lights in just the right places, but when you throw shade on one square, the surrounding squares switch!

Included as a bonus: a button based on the popular and awesome shade button web app. This button plays just the right sound when you say those perfectly insulting words you spent all week thinking of.
